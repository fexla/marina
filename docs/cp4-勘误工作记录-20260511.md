## 1. 第一阶段:修勘误条目

### 输入

`docs/cp4勘误.md` 自由文本 13 条(顺序原样保留):

1. 设置页面滚动条没有正确使用主题
2. 移除"跟随系统主题"功能,不可靠
3. 字体选择改成显示所有已安装字体,然后也需要一些专门的推荐字体
4. 字体选择器的箭头有主题泄露问题
5. 有的时候终端的状态会卡在活跃
6. (开机启动暂未测试 — 仅注记,无须修)
7. 按 Ctrl+F 没有弹出搜索栏,而是显示 `^F`
8. 搜索框没有命中数目
9. 终端搜索的组合键不稳定,Enter 跳上一个
10. Esc 按动不可靠
11. 粘贴多行时没有警告
12. 不要什么都用 Emoji,搞个真正的图标库
13. 导入数据后无法正常渲染,重启恢复
14. Electron Security Warning(Insecure CSP)

> 第 6 条是用户主动注记"暂未测试",不是要修的 bug,我没动它。其余 13 个全部完成。

### 建议接着做的

1. **ipc-protocol.md 升版本**:把 ADR-008 / ADR-009 的实际生效部分一次性改进去:
   - 删 `cmd:session:restart-from-tombstone`、`evt:session:tombstoned` 等不存在的命令/事件

2. **commit 拆分**:目前所有改动都在工作树未 commit。建议你审完后让我按下面的颗粒度提交(可以 git bisect):
   ```
   feat(settings): remove follow-system-theme (ADR-009)
   feat(font): use queryLocalFonts for terminal/UI font picker
   feat(terminal): multi-line paste confirm + search match counter
   fix(terminal): intercept Ctrl+F / Esc via attachCustomKeyEventHandler
   fix(session): schedule initial idle check to avoid stuck-active
   feat(import): in-memory replaceAll instead of app.relaunch (ADR-009)
   ```
